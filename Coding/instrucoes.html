Eletivas de tecnologia

Passo a passo

  1. Duplique a pasta "Template".
  2. Renomeie a pasta para o nome da aula no formato "Aula N - conteúdo".
    Por exemplo, "Aula 1 - laços de repetição".
  3. Abra o arquivo "meu_codigo.js" e trabalhe nele.
  4. Abra o arquivo "index.html" no navegador para ver o resultado.
